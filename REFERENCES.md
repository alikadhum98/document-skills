# مستودعات مرجعية موصى بها

مجموعة مستودعات GitHub مختارة ذات صلة مباشرة بأدوات توليد الوثائق وبيانات المناخ.

---

## توليد وثائق Word

### python-docx-template
**الرابط:** https://github.com/elapouya/python-docx-template  
**الاستخدام:** تصميم قالب Word بالكامل في Microsoft Word مع التنسيق والألوان والجداول، ثم توليد نسخ غير محدودة منه بمتغيرات مختلفة عبر وسوم Jinja2.  
**الفائدة المباشرة:** إنتاج تقارير الجرد الوطني والمذكرات الرسمية من قالب واحد بأرقام محدّثة كل دورة دون إعادة التنسيق.

---

## توليد عروض PowerPoint

### ppt-master
**الرابط:** https://github.com/hugohe3/ppt-master  
**الاستخدام:** توليد شرائح PowerPoint قابلة للتعديل الكامل بأشكال حقيقية لا صور مسطّحة، مع دعم التثبيت عبر Claude Code مباشرة.  
**الفائدة المباشرة:** تطوير مهارة pptx العربية وتحسين جودة الشرائح المولّدة برمجياً.

### ppt-generation (keithmcnulty)
**الرابط:** https://github.com/keithmcnulty/ppt-generation  
**الاستخدام:** استبدال البيانات والرسوم البيانية في قالب PowerPoint موجود لإنتاج نسخ متعددة برمجياً.  
**الفائدة المباشرة:** إنتاج عروض تقديمية للمفاوضات من نفس القالب بتحديث الأرقام فقط.

---

## بيانات الجرد الوطني والمناخ

### ghgdata (sphericalpm)
**الرابط:** https://github.com/sphericalpm/ghgdata  
**الاستخدام:** مجموعة بيانات من تقارير الجرد الوطني لدول أنيكس-1 المقدّمة إلى اتفاقية الأمم المتحدة بصيغة JSON.  
**الفائدة المباشرة:** مقارنة أرقام انبعاثات العراق بدول مشابهة مباشرة دون البحث اليدوي في تقارير UNFCCC.

### open-sustainable-technology (protontypes)
**الرابط:** https://github.com/protontypes/open-sustainable-technology  
**الاستخدام:** دليل شامل للأدوات مفتوحة المصدر في مجالات تغيّر المناخ والطاقة المستدامة.  
**الفائدة المباشرة:** اكتشاف أدوات جاهزة قبل بناء أي أداة من الصفر.
